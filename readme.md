Первое задание папка FilePurifier
 Visual Studio 2022, C#, Avalonia MVVM
 требует: плагин Avalonia for Visual Studio
          nuget пакет CommunityToolkit.Mvvm
		  
Реализована загрузка блоками по 4096 байт что бы можно было обрабатывать большие файлы.
Пакетная обработка файлов, в отдельных потоках из пула.
Быстрый поиск разделителей и пунктуации с использованием SearchValues
Не визуальные классы покрыты тестами.
